Juego del Avión
Este es un juego de avión básico en el que controlas un avión y debes evitar chocar con los enemigos. El objetivo es conseguir la mayor puntuación posible esquivando los enemigos.

Instrucciones de juego
Ejecuta el programa para iniciar el juego.
Utiliza las teclas de flecha izquierda y derecha para mover el avión hacia la izquierda y hacia la derecha, respectivamente.
Evita chocar con los enemigos. Si el avión colisiona con un enemigo, el juego termina.
Intenta conseguir la mayor puntuación posible esquivando los enemigos durante el mayor tiempo posible.
